System Information

Model Details

  • Architecture: FUnIE-GAN (Fast Underwater Image Enhancement GAN)
  • Generator: 5-layer U-Net based architecture
  • Discriminator: 4-layer Markovian discriminator
  • Loss Functions: Adversarial loss + Perceptual loss (VGG-based) + L1 loss
  • Input Resolution: 256Ã—256 pixels

Performance Metrics

  • Processing Speed: ~25+ FPS on edge devices (as reported in paper)
  • Model Size: Lightweight for deployment on AUVs/ROVs
  • Enhancement Capabilities: Color correction, haze removal, contrast enhancement

Maritime Security Applications

  • Threat Detection: Enhanced visibility for identifying submarines, mines, drones, divers
  • Navigation: Improved underwater scene understanding for autonomous vehicles
  • Surveillance: Real-time monitoring of maritime boundaries
  • Search & Rescue: Better visibility for locating objects/persons underwater
